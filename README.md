# Transformer-SNN 低空目标识别

> SRP 项目 — 基于多RIS协同的多源特征数据融合识别与跟踪飞行目标研究（融合识别模块）

面向低空无人机与鸟类的雷达目标识别，提出 Transformer + 脉冲神经网络（SNN）混合架构。以短时观测窗口内的 6×8 多普勒特征矩阵为输入，通过 D1 能量检测、PCA 降维、MLP 注意力融合、统计特征提取，最终由 Transformer 建模全局依赖、LIF 脉冲神经元完成动态决策。

## 识别流程

```
6×8 多普勒特征矩阵
    ↓
D1 能量检测 — 筛除"无目标"
    ↓
PCA 降维（保留 >90% 贡献率）
    ↓
MLP 注意力加权融合
    ↓
统计特征提取（均值/标准差/最大/最小）
    ↓
Transformer Encoder — 全局关联建模
    ↓
SNN (LIF 神经元) — 膜电位累积决策
    ↓
输出：无目标 / 无人机 / 其他飞行物
```

## 关键技术

- **Transformer-SNN 混合架构**：Transformer Encoder 建模统计特征间的全局依赖，LIF 脉冲神经网络作为分类头，通过膜电位时序累积实现动态决策，兼顾全局关联建模与脉冲动态响应
- **代理梯度**：sigmoid 导数近似脉冲函数梯度，解决 SNN 不可导问题
- **LIF 神经元**：膜电位泄漏积分-阈值放电，`τ=2.0, threshold=1.0`
- **可学习位置编码**：替代固定正弦编码
- **决策级统计融合**：提取均值、标准差、最大、最小四维统计量
- **两级识别**：前端能量判据快速筛除 + 后端深度网络精准分类
